# 2d Arrays
 
 1. Definition:-
    ```c++
    //row-wise
    int arr[row][column]={1,2,3,4,....};
    //OR
    int arr[3][3]={{1,2,3},{4,5,6},{7,8,9}};
    ```
 2. To take input:- 
    ```c++
    //Takes row wise input
    cin >> arr[row][column]; //takes O(n^2) time complexity
    ```
 3. To give output:- 
    ```c++
    cout >> arr[i][j]; //takes O(n^2) time complexity
    ```
